## 📌 프로젝트 소개

🎭 본 프로젝트는 **Lies of P**를 모티브로 한 3인칭 액션 RPG 모작입니다.  
다크 판타지와 스팀펑크 요소가 결합된 스타일을 구현하며,  
언리얼 엔진 5.1의 강력한 기능을 활용하여 제작되었습니다.  

본 프로젝트는 **블루프린트**로만 작업되었으며, 별도의 소스 코드는 없습니다.
